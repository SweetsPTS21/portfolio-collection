# Nguyễn Thị Xuân Mai - Summer Sea Case Portfolio

Standalone React portfolio app for `mai-portfolio`.

## UI Style

- Catalog base: `cloud-cosmetic-case` from `catalog-3.md`
- Adaptation: `summer-sea-case`
- Vibe: sea-glass panels, pearl dots, shell compact, soft waves, clean legal-tech polish
- Structure: hero sea case, mirror profile/about, sea shelf projects, conclusion tide board
- Motion focus: wave shimmer, pearl float, glass reveal, card lift, Lenis smooth scroll

## Content Source

All visible portfolio text is derived from `docs/portfolio.md`. The profile image uses `docs/avatar.png`. Six assignment cards open `docs/Bt1.pdf` through `docs/Bt6.pdf`.

## Commands

```bash
npm install
npm test
npm run build
npm run lint
npm run dev
```
