# Lý Bảo Phượng - Interactive Desk Portfolio

Standalone React portfolio app for `phuong-portfolio`.

## UI Style

- Catalog style: `interactive-desk`
- Adaptation: CSS-built personal workspace
- Structure: clickable desk objects for `Giới thiệu`, `Dự án`, and `Tổng kết`
- Motion focus: object lift, spring panel opening, sticky-note float, lamp glow, laptop shine, and animated PDF modal

## Content Source

All portfolio content is derived from `docs/portfolio.md`. Evidence files and avatar come from `docs/`.

## Commands

```bash
npm install
npm run dev
npm run build
```
