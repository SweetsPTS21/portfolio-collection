# Hoàng Thu Trang - Tropical Summer Portfolio

Standalone React portfolio app for `trang-portfolio`.

## UI Style

- Catalog style: `tropical-summer`
- Adaptation: lightweight 2D summer postcard interface
- Structure: summer hero, diary-style About, sticker skills/tools, postcard projects, and conclusion board
- Motion focus: wave drift, cloud drift, sun sparkle, postcard hover lift, and React Motion entrance

## Content Source

All portfolio text is derived from `docs/portfolio.md`. The avatar comes from `docs/avatar.png`.

## Commands

```bash
npm install
npm run dev
npm run build
npm run lint
```
