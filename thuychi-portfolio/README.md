# Thuy Chi — Digital Portfolio

**Theme:** `fairy-mushroom-forest` (Catalog 5)

Portfolio cá nhân với phong cách rừng nấm cổ tích: nấm nhiều màu, tiên nhỏ, đom đóm, lá rụng. Fantasy cute, dreamy, magical.

## Stack

- React 19 + Vite 8
- Ant Design 6 (token system)
- Framer Motion 12 (UI animation)
- Lenis (smooth scroll)
- Tailwind CSS 3 (utility)
- React Router 7

## Scripts

- `npm run dev` — Dev server at :5192
- `npm run build` — Production build
- `npm run preview` — Preview build at :4192
- `npm run test:e2e` — Playwright tests
- `npm run lint` — ESLint
