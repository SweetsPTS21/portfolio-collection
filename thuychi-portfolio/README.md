# Thuy Chi — Digital Portfolio

**Theme:** `underwater-world` (Catalog 5)

Portfolio cá nhân với phong cách thế giới đại dương: xanh dương nhạt, ánh nắng xuyên mặt nước, đàn cá nhiều màu, cua, sao biển, rong biển, san hô, vỏ sò và bubble glass. Theme bám theo Catalog 5 `underwater-world`.

## Stack

- React 18 + Vite 7
- Framer Motion 12 (UI animation)
- Lenis (smooth scroll)
- React Router 6

## Scripts

- `npm run dev` — Dev server at :5192
- `npm run verify:theme` — Underwater catalog compliance check
- `npm run build` — Production build
- `npm run preview` — Preview build at :4192
