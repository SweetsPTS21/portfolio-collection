# Mai Hồng Thắm - Floating Island Portfolio

Standalone React portfolio app for `tham-portfolio`.

## UI Style

- Catalog style: `floating-island`
- Adaptation: lightweight 2D pastel island world
- Structure: hero sky plus floating islands for `ABOUT ME`, skills/tools, `PROJECT`, and `CONCLUSION`
- Motion focus: cloud drift, star twinkle, island float, card lift, and React Motion section entrance

## Content Source

All portfolio text is derived from `docs/portfolio.md`. The avatar comes from `docs/avatar.png`.

## Commands

```bash
npm install --legacy-peer-deps
npm run dev
npm run build
npm run lint
```
