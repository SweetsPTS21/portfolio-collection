# Nghia Portfolio

Theme: `cyberpunk-city` from `ui-styles-catalog.md`.

This portfolio uses a route-based 3 tab structure: Giới thiệu, Dự án, Tổng kết. It is không phải one-page anchor scrolling. The visual system follows the catalog requirements: black cyberpunk base, neon cyan/pink/purple/green, sharp pixel-edge panels, skyline, pixel rain, scanline overlays, glitch text, terminal surfaces, and neon sign project cards.
